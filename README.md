# nousk-plugins
Collection of WordPress plugins focused on development workflow, content structure, and security analysis.

This repository contains a collection of WordPress plugins developed to support real-world workflows, including content management, environment reset, and basic security analysis.

Each plugin is designed with a practical approach, focusing on clarity, usability, and alignment with WordPress standards.

**Plugins included:**
- Nousk CPT Manager: Create and manage custom post types via admin
- Nousk Post Duplicator: Duplicate posts, pages, and custom post types
- Nousk WordPress Reset: Reset WordPress installations for development and testing
- Nousk WP SafeCheck: Security checklist with technical feedback
- Nousk FAQ: Converts structured content into expandable FAQ blocks using simple inline markers
- Nousk AD: Inserting ads into posts and pages with flexible control, supporting manual placement via shortcode and automatic insertion based on paragraph intervals.

These tools are intended for developers, learners, and anyone working with WordPress in development or staging environments.
